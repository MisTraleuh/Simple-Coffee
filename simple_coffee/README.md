## ⚡️ Download | Install

> [Flutter](https://flutter.dev/docs/get-started/install)

## 🚀 Run

```bash
# Clone this repository
$ git clone https://github.com/MisTraleuh/Simple-Coffee.git

# Go into the repository
$ cd Simple-Coffee

# Install the dependencies
$ flutter pub get

# Start the app
$ flutter run
```
